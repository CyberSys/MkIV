to add a new vehicle or unit:
* put all .material files in materialscripts/
* put all .mesh and .skeleton files in models/
* put the .spec file, or make a new one, and put in specifications/
  this file tells the engine how to use the vehicle or unit
* put all images (.png, .jpg, .bmp, .dds, etc) into textures/
* put any audio files into audio/

copyright information for the existing vehicles and units is available in the COPYRIGHT.txt file
